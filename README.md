# Audiobook-to-Speech-Fine-Tuning-ASR-Models-for-Kurdish-Sorani
This project makes a comparison of four automatic speech recognizers (ASR) based on Wav2Vec 2.0 XLSR-53, Whisper Small (multilingual), Whisper Small (razhan/whisper-smallckb), and HuBERT along with their Sorani Kurdish ASR audiobooks..
